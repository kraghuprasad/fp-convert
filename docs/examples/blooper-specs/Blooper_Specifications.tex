%
Following requrements specifications are applicable to the user{-}registration module of the application.%
\begin{enumerate}%
\item%
\hypertarget{ID:259818684}{}%

\hypertarget{ID:1261303219}{}%
Users with an email address and mobile number should be able to create an account in Blooper.%
